Задача 1

Имеется симметричная монета. Напишите функцию генерации независимых случайных величин из нормального и экспоненциального распределений с заданными параметрами.

Часть 1.

1. Напишите функцию uniform генерации случайных величин из равномерного распределения на отрезке [0,1][0,1] с заданной точностью. Это можно сделать, записав случайную величину ξ∼U[0,1]ξ∼U[0,1] в двоичной системе системе счисления ξ=0,ξ1ξ2ξ3...ξ=0,ξ1​ξ2​ξ3​.... Тогда ξi∼Bern(1/2)ξi​∼Bern(1/2) и независимы в совокупности. Приближение заключается в том, что вместо генерации бесконечного количества ξiξi​ мы полагаем ξ=0,ξ1ξ2ξ3...ξnξ=0,ξ1​ξ2​ξ3​...ξn​.

Нужно реализовать функцию uniform так, чтобы она могла принимать на вход в качестве параметра size как число, так и объект tuple любой размерности, и возвращать объект numpy.array соответствующей размерности. Например, если size=(10, 1, 5), то функция должна вернуть объект размера 10×1×510×1×5. Кроме того, функцию coin можно вызвать только один раз, и, конечно же, не использовать какие-либо циклы. Аргумент precision отвечает за число nn. Не используйте np.vectorize и подобные, это медленно и по сути является циклом.

2. Напишите функцию plot_uniform_density. Для U[0,1]U[0,1] сгенерируйте size=200size=200 независимых случайных величин, постройте на одном графике scatterplot (plt.scatter) получившейся выборки (координата yy для всех точек равна нулю), гистограмму по сгенерированным случайным величинам, а также график плотности на отрезке [−0.25,1.25][−0.25,1.25]. Прозрачность точек и гистограммы поставьте равной 0.40.4. Гистограмма должна иметь 1010 бинов, при этом график должен соответствовать плотности (внимательно изучите параметры plt.hist). Для отрисовки плотности используйте вспомогательную функцию uniform_pdf, которая вам доступна (см. шаблон) и представляет собой sps.uniform.pdf, плотность вычисляйте на сетке из 500500 точек. Не забывайте про plt.figure.

3. Напишите функцию plot_uniform_different_precision. Исследуйте, как меняются значения случайных величин в зависимости от precision. Для этого на графике изобразите 66 выборок с разными значениями этого параметра: [1,2,3,5,10,30][1,2,3,5,10,30], используя plt.subplot.

Часть 2.

1. Напишите функцию normal генерации случайных величин в количестве size штук из распределения N(loc,scale2)N(loc,scale2) с помощью преобразования Бокса-Мюллера.

Реализация должна быть без циклов. В качестве параметра size может быть как число, так и объект tuple любой размерности. Старайтесь использовать как можно меньше бросков монеты. Можно использовать реализованную ранее функцию uniform.

2. Напишите функцию plot_normal_density. Для N(0,1)N(0,1) сгенерируйте size=200size=200 независимых случайных величин, постройте график плотности на отрезке [−3,3][−3,3], а также гистограмму и scatterplot по сгенерированным случайным величинам. Все остальные требования аналогичны пункту 22 первой части. Вспомогательная функция для плотности — norm_pdf, представляющая собой sps.norm.pdf.

1. Вы уже научились генерировать выборку из равномерного распределения. Напишите функцию expon генерации выборки из экспоненциального распределения, используя задачу из теории вероятностей:

2. Напишите функцию plot_expon_density. Для Exp(1)Exp(1) сгенерируйте выборку размера size=100size=100 и постройте график плотности этого распределения на отрезке [−0.5,5][−0.5,5], гистограмму и scatterplot по сгенерированным случайным величинам. Все остальные требования аналогичны пункту 22 первой части. Вспомогательная функция для плотности — expon_pdf, представляющая собой sps.expon.pdf.

  • Присылать нужно только код класса с подключением необходимых библиотек. Использовать библиотеки, которых нет в шаблоне, запрещается.

  • Все функции для пунктов с графиками должны возвращать matplotlib.figure.Figure, на котором рисовался график. В шаблоне уже есть пример как это сделать через plt.gcf(), но при этом нельзя вызывать plt.show(), иначе холст сбросится и на проверку пойдёт пустой.
